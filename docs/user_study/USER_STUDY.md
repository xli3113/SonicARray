# 用户测试方案：空间音频算法学习效果评估

## 测试目标

评估 SoundARray 系统是否有助于提升初学者对空间音频算法的认识和理解。

### 研究问题

1. 使用可视化系统后，初学者对空间音频概念的理解是否提升？
2. 交互式可视化是否比传统学习方式更有效？
3. 系统在哪些方面最有助于学习（算法原理、参数影响、实际应用等）？

## 测试设计

### 测试类型

**前后对比测试（Pre-Post Test）**
- 测试前：评估参与者的基础知识水平
- 测试中：使用系统进行学习和实践
- 测试后：评估学习效果和知识提升

### 参与者要求

**目标群体**：初学者
- 对空间音频算法了解有限或完全不了解
- 可能具备基础音频/信号处理知识，但不熟悉空间音频
- 年龄：18-35 岁
- 人数：建议 15-30 人（至少 15 人）

**排除标准**：
- 专业从事空间音频研究或开发
- 已深入理解 VBAP、Ambisonics 等算法
- 无法使用 VR/AR 设备（如严重晕动症）

## 测试流程（总时长：30-45 分钟）

### 阶段 1：前测（8-10 分钟）

#### 1.1 基本信息收集（2 分钟）
- 年龄、专业背景、音频相关经验
- 对空间音频的熟悉程度（1-5 分）
- 是否使用过 VR/AR 设备

#### 1.2 知识测试（6-8 分钟）

**精简版测试**（仅关键题目）

**基础概念题**（5 题，每题 1 分）
1. 什么是空间音频？
2. VBAP 的全称是什么？
3. 空间音频中"增益"的含义是什么？
4. 扬声器阵列的作用是什么？
5. 为什么需要多个扬声器来实现空间音频？

**算法理解题**（3 题，每题 2 分）
1. VBAP 算法的基本原理是什么？（简要说明）
2. 增益平滑的作用是什么？
3. 如何计算声源到扬声器的距离？

**评分标准**：
- 选择题：每题 1 分
- 简答题：0-2 分（完全错误=0，部分正确=1，基本正确=2）
- 总分：11 分

### 阶段 2：系统使用（15-20 分钟）

#### 2.1 系统介绍（3 分钟）
- 快速演示系统功能
- 说明关键可视化元素（扬声器、声源、高亮、连接线）
- 介绍基本操作（移动声源）

#### 2.2 引导任务（12-17 分钟）

**任务 1：基础理解（4 分钟）**
- 将声源移动到不同位置（前、后、左、右）
- 观察哪些扬声器被激活
- 理解距离与增益的关系

**任务 2：算法验证（4 分钟）**
- 将声源放在特定位置（如正前方）
- 预测哪些扬声器应该激活
- 验证预测是否正确

**任务 3：可视化观察（4 分钟）**
- 观察扬声器高度变化与增益的关系
- 观察连接线粗细变化
- 理解可视化与算法的对应关系

**任务 4：对比实验（可选，5 分钟）**
- 将声源放在对称位置
- 对比左右两侧的激活模式

### 阶段 3：后测（7-10 分钟）

#### 3.1 知识测试（5 分钟）
- 使用相同题目（同前测）
- 评估知识提升程度

#### 3.2 快速评估问卷（2-5 分钟）

**核心评估题**（必答，5 分制）
1. 系统帮助我理解了空间音频的基本概念
2. 可视化让我更好地理解了 VBAP 算法
3. 我理解了增益计算的基本原理
4. 系统对理解算法有帮助
5. 愿意继续使用系统学习

**快速反馈**（选答，1-2 分钟）
- 系统最有助于理解哪些方面？（一句话）
- 整体学习体验如何？（一句话）

## 评估指标

### 定量指标

1. **知识提升分数**
   - 后测分数 - 前测分数
   - 提升百分比 = (后测-前测) / 前测 × 100%

2. **各维度得分**
   - 基础概念理解
   - 算法原理理解
   - 实际应用理解

3. **问卷评分**
   - 学习效果平均分
   - 系统可用性平均分
   - 学习体验平均分

### 定性指标

1. **开放式问题分析**
   - 主题提取
   - 常见反馈归纳

2. **访谈内容分析**
   - 学习路径
   - 难点识别
   - 改进建议

   - 可随时退出
   - 不强制完成所有任务

## 预期结果

### 假设
- H1：使用系统后，知识测试分数显著提升
- H2：可视化系统比传统学习方式更有效
- H3：交互式操作有助于理解算法原理

### 成功标准
- 知识提升 ≥ 30%（或提升分数 ≥ 3 分）
- 学习效果问卷平均分 ≥ 4.0/5.0
- 至少 70% 参与者认为系统有帮助
