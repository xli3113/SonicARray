# 用户测试文档

本目录包含 SoundARray 系统的用户测试方案和相关材料。

## 测试方案

- **[USER_STUDY.md](./USER_STUDY.md)** - 主测试方案（30-45 分钟精简版）
  - 测试目标与研究问题
  - 测试设计与流程
  - 评估指标与数据分析

## 测试材料

### 核心材料（必用）

- **[PRE_POST_TEST.md](./PRE_POST_TEST.md)** - 前后测知识测试问卷
  - 精简版：8 题（5 道选择题 + 3 道简答题）
  - 总分：11 分
  - 时间：6-8 分钟

- **[QUICK_EVALUATION.md](./QUICK_EVALUATION.md)** - 快速评估问卷
  - 5 题核心评估
  - 时间：2-5 分钟

- **[INFORMED_CONSENT.md](./INFORMED_CONSENT.md)** - 知情同意书模板
  - 研究说明
  - 参与者权利
  - 隐私保护

### 可选材料（深度分析）

- **[SYSTEM_EVALUATION.md](./SYSTEM_EVALUATION.md)** - 完整系统评估问卷
  - 详细评估（18 题 + 开放式问题）
  - 用于深度分析和研究

- **[DATA_ANALYSIS_TEMPLATE.md](./DATA_ANALYSIS_TEMPLATE.md)** - 数据分析模板
  - 数据收集清单
  - 统计分析模板
  - 结果可视化建议

## 快速开始

1. **阅读测试方案**：[USER_STUDY.md](./USER_STUDY.md)
2. **准备测试材料**：打印或准备在线问卷
3. **执行测试**：按照流程进行
4. **数据分析**：使用模板进行分析

## 测试流程概览

```
前测（8-10分钟）
    ↓
系统使用（15-20分钟）
    ↓
后测（7-10分钟）
    ↓
总计：30-45分钟
```

## 文件说明

| 文件 | 用途 | 时间 |
|------|------|------|
| USER_STUDY.md | 主测试方案 | 参考 |
| PRE_POST_TEST.md | 知识测试 | 6-8 分钟 |
| QUICK_EVALUATION.md | 快速评估 | 2-5 分钟 |
| SYSTEM_EVALUATION.md | 完整评估（可选） | 10-15 分钟 |
| INFORMED_CONSENT.md | 知情同意 | 2 分钟 |
| DATA_ANALYSIS_TEMPLATE.md | 数据分析 | 参考 |

## 注意事项

- 所有测试材料已精简，确保在 30-45 分钟内完成
- 完整评估问卷为可选，用于深度分析
- 建议至少 15 名参与者
- 确保所有参与者签署知情同意书
