# 知情同意书

## 研究项目：SoundARray 空间音频学习系统效果评估

### 研究目的

本研究旨在评估 SoundARray 系统是否有助于提升初学者对空间音频算法的认识和理解。您的参与将帮助我们改进系统，使其更好地服务于教学和学习。

### 研究内容

1. **前测**：评估您对空间音频的基础知识水平（约 30 分钟）
2. **系统使用**：使用 SoundARray 系统进行学习和实践（约 45-60 分钟）
3. **后测**：评估学习效果和知识提升（约 30 分钟）
4. **问卷**：填写系统评估问卷（约 10-15 分钟）

**总时间**：约 2 小时

### 参与要求

- 年龄 18 岁以上
- 对空间音频算法了解有限或完全不了解
- 能够使用 VR/AR 设备（无严重晕动症）
- 愿意完成所有测试环节

### 数据收集

我们将收集以下数据：
- 测试问卷答案
- 系统使用数据（如操作记录）
- 评估问卷反馈
- 访谈内容（如参与深度访谈）

### 隐私保护

- 所有数据将匿名化处理
- 个人信息不会公开
- 数据仅用于研究分析
- 数据将安全存储，仅研究团队可访问

### 您的权利

1. **自愿参与**：您可以随时退出研究，无需说明理由
2. **数据撤回**：在研究完成前，您可以要求撤回您的数据
3. **结果获取**：研究完成后，您可以要求获取研究结果摘要
4. **问题咨询**：如有任何问题，可随时联系研究团队

### 潜在风险

- **最小风险**：本研究不涉及身体或心理伤害
- **设备使用**：VR 设备可能引起轻微不适，如感到不适可随时停止
- **时间投入**：需要约 2 小时时间参与测试

### 潜在收益

- 学习空间音频相关知识
- 体验最新的可视化学习系统
- 为教育技术研究做出贡献
- 获得研究结果反馈（如需要）
